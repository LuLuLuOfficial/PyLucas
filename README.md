# Welcome to PyLucas

---

## 安装

```
pip install PyLucas
```

---

## 支持的功能列表:

- Class
  - LogManager
    - 用于管理日志信息与日志文件.
  - ConfigEditor
    - 用于读取, 编辑配置文件.
  - result
    - 用于作为函数或方法的返回值为编写代码提供便利.
- Function
  - ASCII_Art
    - 用于生成 AsciiArt.
  - GetTimeStamp
    - 获取当前的时间戳.
  - Get_CurrentFrame_Info
    - 获取当前代码段运行的当前帧, 便于定位错误出现位置.
